Determines Manufacturability of a Pallet by looking at all the pieces on the pallet.
Updates the Orion Context Broker with manufacturability info 'CanProccess', 'CanPickupOnly' or 'CannotPickup' for a pallet number.

Is meant to be connected to the IPC's through a webserver.

Usage: ManufacturabilityPallet.exe palletNumber.

Installation : Copy all files to a directory on your pc. Make sure there is a  connection to the webserver and run the executable
