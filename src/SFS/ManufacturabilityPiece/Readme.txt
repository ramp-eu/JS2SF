Determine Manufacturability of a Piece.
Updates the Orion Context Broker with manufacturability info 'CanProccess', 'CanPickupOnly' or 'CannotPickup' for a piece ID.

Is meant to be connected to the IPC's through a webserver.

Usage: ManufacturabilityPiece.exe pieceID

Installation : Copy all files to a directory on your pc. Make sure there is a connection to the webserver and run the executable
